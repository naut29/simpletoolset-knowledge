# Contributing

Thanks for contributing to the Simple Tool Set public knowledge base.

## Steps

1. Choose a file in `categories/`, `tools-md/`, or `use-cases/` to edit, or add a new Markdown file.
2. Ensure the canonical URL line points to `https://simpletoolset.com/...`.
3. Use relative links for files inside this repository.
4. Keep content factual and concise.

## Style rules

- Use clear headings and short paragraphs.
- Include a canonical URL line in each page when applicable.
- Avoid marketing fluff or promotional language.
- Avoid claims that are not supported by the content.

## Link rules

- Canonicals must point to `https://simpletoolset.com/...` only.
- Internal links must be relative paths within this repository.
