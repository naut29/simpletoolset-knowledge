## FAQs

- What's the difference between shades and tints?: Shades are darker variations (color mixed with black), while tints are lighter variations (color mixed with white). Together they create a complete palette.
- How do I use these in a design system?: Copy the CSS variables and add them to your root CSS. Name them semantically (e.g., --primary-100 through --primary-900) for consistent usage.
- Can I adjust the number of shades?: The generator creates 10 standard shades (100-900 scale), matching common design system conventions used by Tailwind CSS and Material Design.
- What color format is outputted?: Colors are output in both HEX and HSL formats. HSL is particularly useful for programmatically adjusting colors.
