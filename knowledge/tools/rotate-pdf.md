## FAQs

- Can I rotate just some pages?: Yes, you can select specific pages to rotate while leaving others unchanged. This is useful when only some pages have orientation issues.
- Which direction does rotation go?: Rotation is clockwise. 90 turns the page right, 180 flips it upside down, and 270 turns it left.
- Will rotation affect PDF quality?: No, rotation simply changes the page orientation metadata. The content quality remains exactly the same.
- Can I undo a rotation?: After downloading, you can upload the PDF again and rotate in the opposite direction. Keep your original file as a backup.
