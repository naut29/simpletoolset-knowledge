## FAQs

- How many colors can I add?: You can add as many color stops as needed for complex gradients. Each stop can have its own position along the gradient.
- What's the difference between linear and radial?: Linear gradients transition in a straight line (with an angle). Radial gradients transition from a center point outward in a circular or elliptical shape.
- Does this generate browser-compatible CSS?: Yes, the generated CSS works in all modern browsers. Vendor prefixes for older browsers can be included if needed.
- Can I use gradients with transparency?: Yes, use RGBA or HSLA colors for transparent color stops. This creates beautiful overlay effects.
