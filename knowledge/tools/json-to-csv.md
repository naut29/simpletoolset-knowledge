## FAQs

- What JSON format is supported?: The converter works with JSON arrays of objects. Each object becomes a row, and object keys become column headers.
- How are nested objects handled?: Nested objects are flattened with dot notation (e.g., 'address.city') or converted to JSON strings depending on complexity.
- Can I convert CSV back to JSON?: This tool focuses on JSON to CSV conversion. For reverse conversion, use a dedicated CSV to JSON tool.
- What happens with arrays inside objects?: Arrays within objects are converted to comma-separated strings or JSON representations to fit the CSV format.
