# User Agent Parser - Decode Browser Information

## Overview

Decode user agent strings to understand browser and device details. Decode user agent strings to identify browser name and version, operating system, device type, and rendering engine.


## Key features

- Browser detection
- OS identification
- Device type recognition
- Engine detection
- Version numbers
- JSON export

## How to use

1. Paste a user agent string into the input
2. Or click 'Use Current' for your browser's UA
3. View parsed browser, OS, and device info
4. Copy the JSON data for logging
5. Test different user agents for debugging

## FAQs

- Can I use my current browser's UA?: Yes, click 'Use Current' to automatically load your browser's user agent string for quick analysis.
- What is a user agent string?: A user agent string is text your browser sends to websites identifying itself - including browser name, version, OS, and device type.
- Can this detect bots and crawlers?: Yes, many bots identify themselves in their user agent. The parser can help identify Googlebot, Bingbot, and other crawlers.
- Why do user agents look so complex?: For historical reasons, browsers include multiple identifiers for compatibility. Modern browsers often include 'Mozilla' for legacy support.
