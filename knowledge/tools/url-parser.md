# URL Parser - Break Down URLs into Components

## Overview

Analyze and break down URLs into components for debugging and understanding. Break down any URL into its components: protocol, host, port, path, query parameters, and hash.


## Key features

- Full URL breakdown
- Query parameter parsing
- Click-to-copy
- Error handling
- Encoded character decoding
- JSON export

## How to use

1. Paste or enter any URL
2. View the parsed components instantly
3. Click any component to copy it
4. Analyze query parameters individually
5. Use for debugging or documentation

## FAQs

- What URL parts are extracted?: Protocol (http/https), hostname, port, pathname, search query string, individual query parameters, hash/fragment, and full origin.
- Does it decode URL-encoded characters?: Yes, URL-encoded characters like %20 (space) and %3D (equals) are decoded for readability while preserving the original encoding option.
- Can I parse UTM parameters?: Yes, all query parameters including utm_source, utm_medium, utm_campaign etc. are extracted and displayed individually.
- What if my URL is malformed?: The parser will show an error message for invalid URLs. Check for missing protocols (http/https) or malformed syntax.
