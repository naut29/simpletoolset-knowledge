# Optimize SVG Icons for the Web

## Overview

SVGs are ideal for icons, but bloated files slow down pages. This guide trims SVGs and checks their contrast before you ship them.

## Key features

- Reduce SVG file size quickly
- Verify icon contrast for accessibility
- Create embeddable base64 snippets
- Keep icon sizing consistent

## How to use

1. Optimize the SVG to remove unnecessary data.
2. Check contrast ratios to ensure readability on backgrounds.
3. Convert to base64 when you need inline embeds.

## FAQs

- Will optimizing change how the SVG looks?: It removes unused data without changing appearance.
- When should I use base64?: Use it for small icons embedded in CSS or HTML.
