# Create Social Media Graphics Quickly

## Overview

When you need visuals fast, start with text-based graphics and polish them for each platform. This guide covers a lightweight workflow for quick publishing.

## Key features

- Generate visuals from short text prompts
- Resize for multiple platform dimensions
- Crop to focal points without distortion
- Compress files for faster sharing

## How to use

1. Create a base graphic with text or a simple image.
2. Resize and crop to match each platform format.
3. Compress and convert to the format you need.

## FAQs

- Do I need separate sizes for each platform?: Yes, each platform has its own aspect ratios.
- Will compression reduce clarity?: Mild compression keeps visuals crisp while shrinking size.
