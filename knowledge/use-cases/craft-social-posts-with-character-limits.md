# Craft Social Posts with Character Limits

## Overview

Different platforms impose strict limits. This workflow helps you keep posts within bounds while staying readable and well formatted.

## Key features

- Count characters and words quickly
- Add spacing for clearer captions
- Create word clouds to review themes
- Keep posts under platform limits

## How to use

1. Draft your caption and check character counts for each platform.
2. Add spacing to improve readability and line breaks.
3. Review key themes with a word cloud before posting.

## FAQs

- Do emojis count as characters?: Yes, most platforms count emojis as characters.
- Should I shorten links in captions?: Short links help keep character counts low.
