# Debug Web Requests and Identifiers

## Overview

When requests fail, context matters. This guide helps you decode tokens, inspect URLs, and analyze request metadata in one place.

## Key features

- Parse URLs to inspect query strings
- Decode JWTs for quick validation
- Interpret user agent strings
- Look up IP metadata for troubleshooting

## How to use

1. Parse the URL to confirm parameters and structure.
2. Decode tokens to inspect headers and payloads.
3. Check user agents and IPs to understand request sources.

## FAQs

- Does decoding a token validate it?: No, decoding shows contents but does not verify signatures.
- Is IP metadata always accurate?: It is approximate and best used for diagnostics.
