# Build a UI Color Palette

## Overview

A good palette reduces design drift and speeds up UI work. This workflow builds consistent shades and checks contrast early.

## Key features

- Generate tints and shades from a base color
- Check contrast ratios for readability
- Create gradients for backgrounds or buttons
- Save consistent color references

## How to use

1. Generate a shade scale from your base brand color.
2. Verify contrast ratios for text and UI components.
3. Create gradients for accents or hero sections.

## FAQs

- How many shades should I create?: Start with 5 to 9 steps for a flexible scale.
- Do I need to check contrast for every color?: Focus on text and interactive elements.
